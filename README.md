# PSMaker123 – Mock Paystub Generator

**PERSONAL / EDUCATIONAL / MOCK USE ONLY**  
A simple console-based mock paystub generator.

**Features**
- Biweekly paystub simulation
- Automatic FICA, federal, and state tax estimation (2026 rates)
- Persistent YTD totals
- Preview before download
- Saves output as plain text files

**Disclaimer**  
This tool is for personal mock/testing/educational purposes only.  
It uses approximate tax rates and calculations — **do NOT** use generated output for taxes, loans, employment verification, or any official purpose.

**Requirements**
- Python 3.6+

python psmaker123.py
